# ABE-ClinicalData
Mediator for saving and querying clinical data with ABE

## Setup
1. Edit the following files:
    * ```nginx/abe-clinicaldata-nginx```
    * ```nginx/abe-clinicaldata-nginx.conf```
    * ```upstart/abe-clinicaldata-upstart.conf```
    * ```config.py```
    * ```install.sh```
    * ```config/config.json```
    * ```config/mediator.json```

2. Run ```install.sh```
    ```
    ./install.sh
    ```
